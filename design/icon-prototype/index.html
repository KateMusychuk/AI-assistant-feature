Implementation note · adaptive icon color

Иконка использует currentColor — она наследует цвет от родительского контейнера.
Логика в коде: если background кнопки светлый → icon color = тёмный (#1f2023); если тёмный → icon color = white.
Это стандартный паттерн в UI Kit (color-contrast() или JS luminance check).
Один SVG файл работает для всех порталов любого цвета.
